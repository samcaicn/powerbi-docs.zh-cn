# <a name="share-a-dashboard-from-the-power-bi-mobile-apps"></a>在 Power BI 移动应用中共享仪表板
适用于：

| ![iPhone](media/mobile-share-dashboard-from-the-mobile-apps/iphone-logo-50-px.png) | ![iPad](media/mobile-share-dashboard-from-the-mobile-apps/ipad-logo-50-px.png) | ![Android 手机](media/mobile-share-dashboard-from-the-mobile-apps/android-phone-logo-50-px.png) | ![Android 平板电脑](media/mobile-share-dashboard-from-the-mobile-apps/android-tablet-logo-50-px.png) | ![Windows 10](media/mobile-share-dashboard-from-the-mobile-apps/win-10-logo-50-px.png) |
|:--- |:--- |:--- |:--- |:--- |
| iPhone |iPad |Android 手机 |Android 平板电脑 |Windows 10 设备 |

使用 [Power BI Pro 许可证](service-free-vs-pro.md)，通过共享 Power BI 移动应用中的链接，邀请同事查看仪表板。 你与之共享仪表板的人员必须拥有和你相同的电子邮件域，还要有一个 Power BI Pro 许可证，或者内容需要位于[高级容量](service-premium.md)中。

从适用于 iOS 和 Android 的 Power BI 移动应用中，还可以[批注并共享磁贴、报表或视觉对象的快照](mobile-annotate-and-share-a-tile-from-the-mobile-apps.md)。 

## <a name="share-a-dashboard-from-your-iphone"></a>在 iPhone 中共享仪表板
1. 在仪表板底部的操作栏中，点击“共享”图标 ![“共享”图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-share-dashboard-icon.png)。
   
   ![iPhone 共享图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-dashboard-invite.png)
2. 点击“**邀请同事**”。
3. 键入名称（用逗号隔开）和随附在仪表板邀请中的消息。
4. 若要允许再次共享，请选中**允许收件人共享此仪表板**选项。
   
   允许再次共享后，你的同事可以通过 Web 或移动应用向组织中的其他人转发此电子邮件邀请。
5. 点击右上角的**发送**。
   
   你的同事将收到包含仪表板的直接链接的电子邮件邀请。 该邀请将在一个月后过期。 他们在浏览器或 Power BI 移动应用中打开该邀请时，邀请会添加到其 Power BI 帐户的“与我共享”部分。
   
   ![与我共享](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   详细了解[与同事共享仪表板的相关说明](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-iphone"></a>在 iPhone 中取消共享仪表板
如果你是仪表板所有者，只可以取消共享仪表板。

1. 在仪表板底部的操作栏中，点击“共享”图标 ![“共享”图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-share-dashboard-icon.png)。
2. 点击“**邀请同事**”。
   
   使用下列短语，可以看到你已经和其共享仪表板的同事列表：
   
   * **可以查看**：他们可以查看仪表板，但不能共享它。
   * **可以查看和再次共享**：他们可以查看仪表板，并且与其他同事共享。
3. 点击**编辑**。
   
    ![“编辑”图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-edit-invite-dashboard.png)
4. 若要停止共享，请点击名称旁边的红色圆圈，然后点击**删除**。

## <a name="share-a-dashboard-from-your-ipad"></a>在 iPad 中共享仪表板
1. 点击仪表板右上角的“共享”图标 ![“共享”图标](media/mobile-share-dashboard-from-the-mobile-apps/pbi_ipad_shareiconblk.png) 。
2. 点击**邀请同事**，然后键入电子邮件地址以及随附在仪表板邀请中的消息。
3. 若要允许再次共享，请选中**允许收件人共享此仪表板**选项。
   
   > [!NOTE]
   > 允许再次共享后，你的同事可以通过 Web 或移动应用向组织中的其他人转发此电子邮件邀请。
   > 
   > 
4. 点击右上角的**发送**。
   
   你的同事将收到包含仪表板的直接链接的电子邮件邀请。 该邀请将在一个月后过期。 他们在浏览器或 Power BI 移动应用中打开该邀请时，邀请会添加到其 Power BI 帐户的“与我共享”部分
   
   ![与我共享](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   详细了解[与同事共享仪表板的相关说明](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-ipad"></a>在 iPad 中取消共享仪表板
如果你是仪表板所有者，只可以取消共享仪表板。

1. 点击仪表板右上角的“共享”图标 ![“共享”图标](media/mobile-share-dashboard-from-the-mobile-apps/pbi_ipad_shareiconblk.png) 。
   
   可以看到你已经和其共享仪表板的同事列表。 在他们的名称下面是以下词语：
   
   * **等待**：他们尚未查看仪表板。
   * **可以查看**：他们可以查看仪表板，但不能共享它。
   * **可以查看和再次共享**：他们可以查看仪表板，并且与其他同事共享。
2. 点击**编辑**。
3. 若要停止共享，请点击名称旁边的红色圆圈，然后点击**删除**。

## <a name="share-a-dashboard-from-your-android-device"></a>在 Android 设备中共享仪表板
1. 在仪表板的主页上，点击省略号 (...)，然后点击**“邀请同事”**。
   
   ![邀请同事](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-tablet-share-dashboard.png)
2. 或者，在仪表板的右上角点击“邀请”图标 ![邀请图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-invite-icon.png)。

    如果你是仪表板所有者，你将看到与之共享此仪表板的同事的列表，并附有以下说明：

    -   **可以查看**：他们已经查看了仪表板，但不能进行共享。

    -   **可以查看和邀请**：他们已查看仪表板，并且可以与其他同事共享。

1. 键入电子邮件地址以及仪表板邀请随附的消息。 否则 Power BI 会发送默认消息。
2. 若要允许再次共享，请选中**允许收件人共享此仪表板**选项。
   
   > [!NOTE]
   > 允许再次共享后，你的同事可以通过浏览器或移动应用向组织中的其他人转发此电子邮件邀请。
   > 
   > 
3. 点击“发送”图标 ![“发送”图标](media/mobile-share-dashboard-from-the-mobile-apps/pbi_andr_sendplane.png) 发送邮件。
   
   你的同事将收到包含仪表板的直接链接的电子邮件邀请。 该邀请将在一个月后过期。 他们在浏览器或 Power BI 移动应用中打开该邀请时，邀请会添加到其 Power BI 帐户的“与我共享”部分
   
   ![与我共享](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-shared-with-me-left-nav.png)
   
   详细了解[与同事共享仪表板的相关说明](service-share-dashboards.md)。

### <a name="unshare-a-dashboard-from-your-android-device"></a>在 Android 设备中取消共享仪表板
只有仪表板所有者可以取消共享仪表板。

1. 在仪表板的右上角点击“邀请”图标 ![邀请图标](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-android-invite-icon.png)。 
   
   可以看到你已经和其共享仪表板的同事列表。
2. 若要停止与同事共享仪表板，请点击名称旁边的 **X** \> **删除**。

## <a name="share-a-dashboard-from-your-windows-10-device"></a>在 Windows 10 设备中共享仪表板
1. 在仪表板中，点击“邀请”图标 ![邀请图标](media/mobile-share-dashboard-from-the-mobile-apps/pbi_andr_inviteicon.png)。
   
   或者，在仪表板主页上，右键单击或按住一个仪表板，然后点击“邀请”。
   
   ![邀请](media/mobile-share-dashboard-from-the-mobile-apps/pbi_win10_sharedash.png)
   
   如果你是仪表板所有者，你将看到与之共享此仪表板的同事的列表，并附有以下说明：
   
   **只读**；他们已经查看了仪表板，但不能进行共享。
   
   **阅读并重新共享**：他们已查看仪表板，并且可以与其他同事共享。
2. 键入电子邮件地址以及仪表板邀请随附的消息。 否则 Power BI 会发送默认消息。
   
   ![“邀请”对话框](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-windows-10-share-dashboard.png)
3. 若要允许再次共享，请选中**允许收件人共享此仪表板**选项。
   
   > [!NOTE]
   > 允许再次共享后，你的同事可以通过浏览器或移动应用与组织中的其他人共享此仪表板。
   > 
   > 
4. 点击“发送”图标 ![“发送”图标](media/mobile-share-dashboard-from-the-mobile-apps/pbi_win10ph_sendicon.png)。
   
   你的同事将收到包含仪表板的直接链接的电子邮件邀请。 该邀请将在一个月后过期。 他们在浏览器或 Power BI 移动应用中打开该邀请时，邀请会添加到其 Power BI 帐户的“与我共享”部分
   
   ![与我共享](media/mobile-share-dashboard-from-the-mobile-apps/power-bi-iphone-shared-with-me-left-nav.png)
   
   详细了解[与同事共享仪表板的相关说明](service-share-dashboards.md)。
